To build this project you need to have Java installed

To setup the project, including getting the Antlr binaries, just run `setup.bat`

To build this project download [Antlr4](https://www.antlr.org/download.html) and put it into the project root directory.
Depending on the Version of your Antlr-JAR-File you may need to adjust the generate script in `package.json`
